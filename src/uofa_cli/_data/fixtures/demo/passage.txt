Demo passage — UofA conference-floor evaluation fixture.

We assessed the credibility of a centrifugal blood pump CFD model
against an in-vitro flow benchmark. Numerical code verification was
performed at level 2 against a published analytical pipe-flow solution.
Discretization error was characterized at level 3 via a three-mesh
grid convergence study. Output comparison evaluated steady-state
pressure-volume curves against bench measurements at level 3, with the
predicted operating point lying within the acceptance band.

Decision: accepted for use as supporting evidence in the COU defined
below. This bundle is a demonstration artifact only — not a real
clinical or regulatory submission.
