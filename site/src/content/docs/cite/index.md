If you reference UofA in published work, please cite the project repository for now. A formal citation will be added after the praxis defense (target October 2026).

## Pre-publication citation

```bibtex
@software{vettrivel_uofa_2026,
  author  = {Vettrivel, Vishnu},
  title   = {Unit of Assurance: Machine-Verifiable Credibility Evidence for Regulated Computational Simulation},
  year    = {2026},
  url     = {https://github.com/cloudronin/uofa},
  note    = {Doctor of Engineering praxis in progress, George Washington University}
}
```

## Conference appearance

If you reference the NAFEMS Americas 2026 talk specifically:

> Vettrivel, V. (2026, May 27). *The Unit of Assurance: Machine-Verifiable Credibility Evidence for Simulation-Based Decisions in Regulated Industries.* NAFEMS Americas Conference, St. Charles, MO.
