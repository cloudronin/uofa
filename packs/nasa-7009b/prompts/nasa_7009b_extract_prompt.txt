NASA-STD-7009B defines 19 credibility factors. The first 13 are shared with ASME V&V 40-2018, and 6 are NASA-specific:

  1. Software quality assurance
  2. Numerical code verification
  3. Discretization error
  4. Numerical solver error
  5. Use error
  6. Model form
  7. Model inputs
  8. Test samples
  9. Test conditions
  10. Equivalency of input parameters
  11. Output comparison
  12. Relevance of the quantities of interest
  13. Relevance of the validation activities to the COU

NASA-only (set factorStandard to "NASA-STD-7009B"):
  14. Data pedigree
  15. Development technical review
  16. Development process and product management
  17. Results uncertainty
  18. Results robustness
  19. Use history

For each factor, extract:
- factorType: one of the 19 names above (exact match required)
- factorStandard: "ASME-VV40-2018" or "NASA-STD-7009B" as appropriate
- assessmentPhase: "capability" or "results" (required for all NASA factors)
- factorStatus: "assessed" or "not-assessed"
- requiredLevel: integer 0-4 (NASA CAS scale, if assessed)
- achievedLevel: integer 0-4 (NASA CAS scale, if assessed)
- hasEvidence: IRI(s) to backing evidence (ReviewActivity, ProcessAttestation, etc.)
- acceptanceCriteria: the acceptance criteria text (if assessed)
- rationale: the justification text (if assessed)
